**NOTE: This release introduces changes to preview sizes.**

- Added `grand` (1280x1280px)
- Discontinued `small` (100x100px) and `small_125` (125x125px)
- Extended bounding rectangle of `large` from 620x500px to 620x620px. Same for `x_large`: formerly 1024x768px, now 1024x1024px 
- Preview images are no longer generated above the original file size. As a result, __only `maximum` (the original size) is guaranteed to exist__. The UI already falls back gracefully, but API consumers and direct media shortcut users (`/entries/ID/image.jpg?resolution=SIZE`) must handle missing sizes themselves.
- These changes are applied to newly uploaded media only (no recalculation of previews for existing media)
